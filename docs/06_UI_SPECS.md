## 📱 1. プロジェクト一覧画面

### 機能
- プロジェクト一覧表示（作成日降順）
- 新規プロジェクト作成ボタン
- 各プロジェクトのステータス表示
- プロジェクト詳細へのリンク

### ステータス表示色
| ステータス | 色 | アイコン |
|-----------|-----|---------|
| created | グレー | 📝 |
| uploaded | ブルー | 🎵 |
| transcribing | ブルー（アニメーション） | 🔄 |
| transcribed | グリーン | ✅ |
| formatting | ブルー（アニメーション） | 🔄 |
| formatted | グリーン | ✅ |
| generating_images | ブルー（アニメーション） | 🎨 |
| completed | グリーン | ✅ |
| failed | レッド | ❌ |

## 📄 2. プロジェクト詳細画面

### Phase構成
```
【Phase 1: アップロード】
  [ファイルを選択] または ドラッグ＆ドロップ
  対応形式: MP3, WAV, M4A, OGG (最大25MB)

【Phase 2: 文字起こし】
  [文字起こし実行]
  
【Phase 3: 整形・シーン分割】
  [整形・分割実行]
  
【Phase 4: 画像生成】
  [全生成] [未生成のみ] [失敗のみ]
  
  ┌──────────────────────────────────────────┐
  │ シーン 1: タイトル (role)                │
  ├──────────────────────────────────────────┤
  │ セリフ: [dialogue]                        │
  │ 要点: • [bullet1] • [bullet2]            │
  │ プロンプト: [編集可能]                    │
  │ [生成] [再生成]                           │
  │ [生成画像プレビュー]                      │
  └──────────────────────────────────────────┘
  
【Phase 5: ダウンロード】
  [画像ZIP] [セリフCSV] [全ファイルZIP]
```

### シーンカード詳細

**ボタン表示条件:**
| 状態 | 表示ボタン |
|------|-----------|
| 未生成 | [生成] |
| 生成中 | スピナー |
| 完了 | [再生成] ＋ 画像プレビュー |
| 失敗 | [失敗再試行] ＋ エラーメッセージ |
| ポリシー違反 | エラーメッセージ ＋ プロンプト編集促進 |

**一括操作:**
- **[全生成]**: 全シーンの画像を生成
- **[未生成のみ]**: 未生成シーンのみ
- **[失敗のみ]**: 失敗シーンのみ
